Sobre GitHub

O GitHub é uma plataforma web que utiliza o Git para gerenciar projetos de software. Ele fornece recursos adicionais, como controle de acesso, colaboração e hospedagem de repositórios. Aqui está um exemplo de como criar um repositório no GitHub:

  1. Faça login no GitHub.
  2. No canto superior direito, clique no sinal "+" e escolha "Novo repositório".
  3. Preencha as informações do repositório e clique em "Criar repositório".
  4. Se desejar, siga as instruções para enviar código para o repositório.
